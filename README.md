# Home Assistant Blueprints

This repository contains a collection of scripts and automations developed for my personal Home Assistant setup.

You are free to use, modify, and adapt these configurations for your own environment.

## How to Use
1. Navigate to the specific blueprint file (`.yaml`).
2. Click the **Raw** button to get the direct URL.
3. In Home Assistant, go to **Settings** > **Automations & Scenes** > **Blueprints**.
4. Click **Import Blueprint** and paste the URL.
